# Fictional TalkTrace Cases

These examples are fictional. They show the intended workflow without exposing real chat records.

## 1. 求安慰

```text
对象：小A
关系：普通朋友，最近想拉近一点
对方：最近真的好累，什么都不想干
我的感觉：想安慰，但怕太用力
```

我判断：这更像求接住，不一定是在求解决方案。

别这样回：直接讲道理、逼对方振作、过度心疼。

可以发：

1. 稳妥版：“听起来是累到连处理事情的力气都没了。今天先别逼自己了。”
   效果：接住情绪，不越界。
2. 拉近版：“那你先瘫一会儿，我在。想说就丢给我，不想说也没事。”
   效果：给陪伴感，但不逼问。
3. 收尾版：“先躺着吧，今天不用强撑。”
   效果：短、稳、不会增加负担。

## 2. 展示/表演

```text
对象：小B
关系：熟一点的网友
对方：我今天居然把那个拖了三周的东西弄完了
我的目标：接住对方的小得意
```

我判断：对方是在展示完成感，想被看见。

别这样回：只说“不错”、马上问下一个任务、泼冷水。

可以发：

1. 稳妥版：“这个确实该夸一下，拖了三周还能收回来挺不容易。”
   效果：承认成果。
2. 拉近版：“可以啊，终于把这个债还了，今晚能安心一点了吧。”
   效果：带一点熟人感。
3. 收尾版：“行，这个值得奖励自己一下。”
   效果：轻松结束。

## 3. 调侃/玩梗

```text
对象：小C
关系：朋友
对方：我现在像一个被生活腌入味的咸菜
我的目标：接梗，不要突然严肃
```

我判断：这是自嘲式玩梗，先接住比安慰更合适。

别这样回：突然深度共情、认真分析压力来源、只回“哈哈哈”。

可以发：

1. 稳妥版：“那你现在应该已经很下饭了。”
   效果：接梗。
2. 拉近版：“别腌太久，差不多捞出来晾一下。”
   效果：玩笑里带一点关心。
3. 收尾版：“咸菜老师辛苦了。”
   效果：短促轻松。

## 4. 试探

```text
对象：小D
关系：暧昧边缘，不确定
对方：你应该也没多想我吧
我的目标：不装冷，也不一下子过度表态
```

我判断：这可能是在试探你是否在意，但不确定。

别这样回：反问审问、强行表白、装作完全不在意。

可以发：

1. 稳妥版：“也不是没想，只是没太确定你想不想听。”
   效果：承认一点在意，同时留余地。
2. 拉近版：“你这句问得就像在钓我说实话。”
   效果：接住试探，轻微暧昧。
3. 收尾版：“有想一点。”
   效果：短、直接、不过量。

## 5. 冷场复盘

```text
对象：小E
关系：普通朋友
我发：那你自己看着办吧
对方反馈：之后没有继续回
我的目标：复盘为什么变冷
```

我判断：这句话可能被读成撤退或不耐烦，不一定是对方故意冷你。

复盘结论：

- 如果当时目标是收尾，这句话有效但偏硬。
- 如果目标是维持连接，可以换成：“那你按舒服的来，后面想说再跟我讲。”
- 如果这类反馈重复出现，可以更新该对象 `style.md`：面对小E，收尾时避免“你自己看着办”这种容易显得切断的话。
